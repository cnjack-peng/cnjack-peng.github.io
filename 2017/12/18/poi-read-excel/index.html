POI两种读取Excel的方式

发表于 2017-12-18 | 分类于 软件开发

前言

在实际项目上遇到了上传 Excel 并解析到数据库中的需求，经分析后选择 Apache POI 来实现，故事也就开始了……

常规模式

优点：取值方便，可供调用方法多；

缺点：一次性读完值，在读取大量数据时可能会出现堆溢出。

事件驱动模式

优点：逐个单元格读取，理论上多大数据都能处理；

缺点：调用方法较少，需要自己动手堆数据进行处理。
